# Nodejs-simpleBlog
Node+Express+MongoDB 简单博客系统

## 目录说明
+ models：数据库模型文件目录
+ public：公共文件目录（css、js、img、ueditor等）
+ routers：路由文件目录
+ schemas：数据库结构文件（schema）
+ views：模板视图文件目录
+ app.js：应用（启动）入口文件
+ package.json：版本、依赖等
+ 此外还应该在此根目录下创建一个“db”文件夹作为数据库存储目录，node init会根据package.json文件创建一个node_modules文件夹并安装所需依赖
